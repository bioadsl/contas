# Upgrade guide for Calendar v4.0.4

The following properties were made private in DayCellRenderer#dayCellRenderer and AgendaView#dayCellRenderer, relying on
their existence not recommended:

* `key`
* `data`
* `visibleColumnIndex`
* `isOtherMonth`
* `visible`
* `tomorrow`
* `isRowStart`
* `isRowEnd`
* `renderedEvents`
